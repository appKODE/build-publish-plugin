# Migration guides

Pick the guide that matches where you're coming from:

| You are on… | Upgrading to… | Guide |
| --- | --- | --- |
| The old, non-`novo` `build-publish` plugin | any `build-publish-novo` release | [Legacy → build-publish-novo](./legacy-to-novo.md) |
| A `build-publish-novo` **1.x** release | **2.0** | [Migrating to 2.0](./v2.md) |
| A `build-publish-novo` **2.0** release | **2.1** | [Migrating to 2.1](./v2.1.md) |

If you're jumping from the legacy plugin straight to 2.1, do the
[legacy → novo](./legacy-to-novo.md) migration first, then apply the
[2.0 changes](./v2.md) and finally the [2.1 changes](./v2.1.md).
